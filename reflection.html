Responsive design is to create websites which can adjust and looks good on the devices that they are currently viewed on. In todayâ€™s world, internet is one of the most powerful forms of media. The sharing of internet using desktops, mobile devices, tablets is growing rapidly. So, it is compulsory to make a website or any application that works on any devices and satisfy the client needs. In particular, web technology is in a full need of responsive web designs. People using mobile devices may feel less comfortable in using the web application which are designed for desktop computers. So, the new frameworks like React, Angular, Vue will be helpful to make the upcoming or old websites fully responsive by making many changes.
With the responsive design, the content of website will be same in all devices but the flexibility, alignment, screen size will be different according to the resolution, size of the devices. In simple terms, the horizontal view in desktop browser will becomes vertical view with scroll bar in mobile or tablet devices so that the user can read the site by scrolling the nav bar. Also, the sharing would be easy among mobile devices when the page is fully responsive, and the most important information is displayed to the user when the screen size is restricted. I have learnt a lot about responsive web design during my course. Using media queries, I learnt to resize the body section elements according to the screen sizes. Usage of chrome developer tools in testing the responsive design and compiling the code is very helpful. Moreover, I learnt to style the navbar in the form of hamburger menu using the CSS. In this way, a lot of space will be saved for mobile devices, and it also creates the user-friendly experience.
